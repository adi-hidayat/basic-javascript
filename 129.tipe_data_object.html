Associative array
- di bahasa pemrograman lain seprti php, kita bisa menggunakan index berupa tipe data lain setelah number, misal string
- fitur ini biasanya disebut associative array tidak didukung.
- di javascript associative array tidak didukung
- jika kita memaksa memasukkan data bukan number di index array
maka javascript akan merubah tipe data array tersebut menjadi object, dan ini bisa berbahaya karena beberapa operasi di array mungkin bisa berubah hasilnya

Tipe Data Object
- Tipe data object adalah tipe data yang mirip dengan tipe data array
- Yang membedakan adalah index pada tipe data object bisa menggunakan string
- index di object biasanya disebut attributes atau properties, bukan index
