In Operator - In operator adalah operator yang bisa dilakukan untuk mengecek apakah sebuah properti ada di dalam object atau tidak - jika properti tersebut ada di object, maka hasilnya true, sedangkan jika tidak ada maka hasilnya false - tidak hanya di object, in juga bisa digunakan untuk mengecek index di array In operator mengecek properti ada atau tidak, meskipun nilai dari properti adalah null atau undefined maka tetap true jika properti tersebut tersedia dalam sebuah objects
