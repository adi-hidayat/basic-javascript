Falsy


- Falsy atau kadang ditulis falsey, adalah value yang ketika dalam konteks boolean, dia dianggap false
- ini adalah salah satu fitur unik dari javascript yang berguna namun kadang juga sering membingungkan
- jadi di javascript kondisi itu tidak hanya bisa boolean, tapi dluar boolean pun bisa. namun kita harus tahu beberapa data falsy atau dianggap false

contoh data false :
false = false
0, -0 = false
"", '', `` = false
null = false
undefined = false
NaN = false
